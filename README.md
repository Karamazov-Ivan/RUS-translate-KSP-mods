# RUS-translate-KSP-mods

Неофициальная локализация (Русификаторы KSP модов) некоторых модов

Переводы выполнены через DeepSeek, без особого внимания к деталям и служат лишь удобству автора.

Перевод актуален на момент выхода.

Скачать и скопировать в папку GameData

Перевод модов KSP:

- Antenna Helper Next
- Contract Configurator
- Dynamic Battery Storage
- Kerbal Engineer (только подсказки (?))
- Near Future Exploration
- OPT (Только описание предметов. При попытке изменить названия предметов ломает RasterPropMonitor)
- SCANsat
